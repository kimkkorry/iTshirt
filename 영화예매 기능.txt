영화 예매 기능 - 김태일
- 사용자
 - 영화 예매
  - 영화별 조회 기능
    - 등록된 영화들 중 하나를 고르면 시간 및 예약 가능한 좌석 수가 프린트
    - 시간까지 선택하면 좌석들이 보여지며 현재 예약된 좌석 및 선택한 좌석이 구분되어 보여짐
  - 영화 예매 시 예매 내역 조회와 취소 때 사용하기 위한 이름, 전화번호와 같이 개인 정보를 입력받음
  - 카드와 현금 구분해서 결제 가능
  - 회원이라면 포인트 및 쿠폰 사용 가능, 결제 후 포인트 적립 기능
 - 예매 내역 조회
  - 개인 정보 이용하여 몇 관, 무슨 영화, 몇시 부터 몇시까지 상영하는지, 어떤 좌석인지 조회 가능
 - 예매 취소
  - 입력한 개인정보 이용하여 예매 취소 기능 구현
 - 회원가입
  - 회원으로 로그인 시 포인트 및 쿠폰 적립, 사용 위해
 - 포인트 및 쿠폰 조회
  - 포인트 사용 내역 및 현재 보유한 포인트 조회 기능, 현재 보유한 쿠폰 조회 기능


- 관리자
 - 영화 및 상영관 추가
  - 영화이름과 런타임, 시작 시각을 넣으면 하루치 영화 상영 시간표 자동 생성
 - 관별 예약 내역 조회
  ex) 1관 해리포터 2/72
 - 좌석별 예약 내역 조회
  ex) 사진처럼 예약 좌석이 보여지게끔
 - 예매내역 초기화 및 파일 생성
  - DB가 아닌 file기반이라 파일 생성및 초기화를 해주어야함
